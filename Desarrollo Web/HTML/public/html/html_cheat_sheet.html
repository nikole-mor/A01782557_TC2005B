HTML Cheat Sheet

To create a page in HTML there are several tags that help us with the structure and content of the page.

Tags

a- Hyperlinks

The anchor element, with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

Content within each a should indicate the link's destination.

For example:

Find the GIF you want here: Explore GIFs

p- Paragraph

Defines a paragraph of text.

For example:

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Quo, doloribus aliquid. Est iure eum laborum temporibus harum sunt nemo, accusamus nulla numquam sit veniam incidunt eveniet maxime tenetur esse eos!

h1, h2, h3,...,- Headings

Define HTML headings, with "h1" being the highest or the biggest letter level and "h6" the lowest.

For example:

Header h1

Header h2

Header h3

Header h4

Header h5
Header h6

div- Divisor

Defines a division or a section, often used as a container for other elements to style them with CSS.

For example:

This text is in a div section that is styled later with the CSS

Span

Used to group inline elements in a document. We can use this element to wrap sections of text for styling purposes or to add attributes to a section of text without creating a new line of content

For example:

In this text, we use span to highlight a part of the text.

Strong

Defines text with strong importance, typically rendered as bold.

For example:

This is important text

i- Italic

Defines a section of text in an alternate voice or mood, typically rendered in italics.

For example:

This text is italicized.

u- Underline

"u" tag defines underlined text and ins tag defines inserted text

For example:

Underlined text

b- Bold

Defines bold text without any extra importance

For example:

This text is bold.

Strike (Strikethrough)

"strike" defines a strike or line through Text. This tag creates a cut line in the text.

For example:

This text is struck through.

Small

Defines smaller text.

For example:

This is small text.

em

Defines emphasized text, typically rendered in italics.

For example:

This text is emphasized.

Header

Represents a container for introductory content or a set of navigational links.

For example:

This is a header

Section

"section" is a semantic element that enables you to create standalone sections on a web page

Defines a section in a document, often with a heading

For example:

Section 1

Content

Section 2

Content

Article

Defines independent, self-contained content that can be distributed independently, such as a blog post or a news article.

For example:

This is an article

Here is the text of the article

Aside

represents a section of a page that consists of content that is tangentially related to the content around the aside element, and which could be considered separate from that content.

For example:

This is an aside content.

Footer

Defines a footer for a document or a section, typically containing information about the author, copyright information, or navigation links.

For example:

Contact info: 5510927461

Nav

represents a section of a page that links to other pages or to parts within the page

For example:

Inside the same page:

Home

To external pages:

Google

Defines a thematic break in the content, often rendered as a horizontal line.

For example:

Paragraph after the horizontal line.

ul- Unordered List

Defines an unordered (bulleted) list

For example:

  • Element 1
  • Element 2
  • Element 3

ol- Ordered List

Defines an ordered (numbered) list.

For example:

  1. Element 1
  2. Element 2
  3. Element 3
